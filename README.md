# Techgy

Techgy là một mẫu website blog nhiều trang miễn phí được xây dựng bằng Bootstrap 5, HTML5 và CSS3. Mẫu website blog công nghệ này chào đón người truy cập với một phần hero hiện đại, kết hợp cùng bảng màu chuyên nghiệp. Giao diện sở hữu nhiều thành phần UI phong phú như: thanh điều hướng trên cùng, nút kêu gọi hành động (CTA), hiệu ứng khi rê chuột, carousel bài viết nổi bật, khu vực blog với nhiều layout, giao diện đăng ký nhận bản tin, điều hướng footer, giao diện form liên hệ, trang chi tiết bài viết, trang tác giả, trang danh mục, trang tìm kiếm, trang đăng nhập/đăng ký và nhiều thành phần khác.

Tất cả các thành phần đều 100% responsive và hiển thị mượt mà trên mọi thiết bị, không phụ thuộc vào kích thước màn hình.
